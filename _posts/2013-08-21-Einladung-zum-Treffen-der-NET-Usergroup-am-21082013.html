--- layout: postnew title: "Neue Features von ASP.NET" categories: treffen date: 2013-08-21 18:00:00 speaker: Hannes Preishuber place: T-Systems MMS Dresden placelink: https://goo.gl/maps/cXLm4 --- Nach einer Sommerpause findet das nächste Treffen unserer Usergroup am 21.08.2013 statt. Wir treffen uns bei der T-Systems MMS und thematisch geht es um einige der neuen Features von ASP.NET. Im Anschluss daran gibt es wieder Gelegenheit zum netzwerken bei einem Getränk. Die Gästeliste zur Planung findet ihr hier. Abstract

Die mobile Nutzung des Internet wächst rasant. Es stellen sich Fragen nach Plattform und Device Unterstützung. Dieser Abend widmet sich diesem Aspekt aus Web Developer Sicht. Wie kann man mit Microsoft ASP.NET Technologie so einfach wie möglich Web Apps bauen. Dabei wird Hannes Preishuber sehr pragmatisch vorgehen. Themen die angeschnitten werden User Experience, Usablity, Server vs Client, Responsive Design, Webforms Rendering, JQuery, JQuery mobile, Performance, Offline Storage. Im speziellenwerden neue Funktionen von ASP.NET 4.5 verwendet.

Über den Sprecher

Der Referent Hannes Preishuber beschäftigt sich dem Thema Web weit über eine Dekade. Er hat mehrere Bücher zu ASP.NET verfasst, war MVP und Sprecher u.a. auch auf der TechEd. Darüber hinaus ist er Gründer und Vorstand des Microsoft Gold Learning Partners ppedv.
