Thema 1: Data Quality Services (Matthias Wendt)

Die Data Quality Services im SQL Server 2012 sind eine neue Teilkomponte und stellen ein integriertes Datenqualitätswerkzeug von Microsoft dar. Mittels einer Wissensdatenbank lassen sich verschiedenste Maßnahmen zur Steigerung der Datenqualität im Unternehmen durchführen, wie z.B. Cleansing, Matching oder Data Profiling. Die Nutzung kann dabei manuell über den DQS-Client oder automatisiert in einem SSIS-Task geschehen. Der Vortrag soll einen Einblick in die Nutzung der DQS und möglicher Anwendungsfälle geben.

Thema 2: Einstieg in die neuen TSQL-Features von SQL Server 2012 (Martin Hey)

Mit SQL Server 2012 sind ein paar neue und auch interessante Features hinzugekommen. Wir werden uns mit ein paar neuen Funktionen beschäftigen, die nun verfügbar sind und herausfinden, was damit alles möglich ist. Interessant ist das bestimmt auch für all diejenigen, die das Treffen der SQL-Server-Gruppe im Juni verpasst haben.
